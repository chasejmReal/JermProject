## SETUP DJANGO (for Windows) ##
1. Open up terminal enviroment 

2. type: 
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

3. type (not needed): 
.\myenv\Scripts\activate

4. type: 
pip install django

5. type: 
cd ProjectSigma

6. type: 
python manage.py migrate

7. type (if needed): 
python manage.py runserver
